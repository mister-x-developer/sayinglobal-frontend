# SAYIN GLOBAL - Frontend

## Premium Digital Livestock Marketplace

### Quick Start

```bash
# Install dependencies
npm install

# Setup environment
cp .env.example .env.local

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

### Tech Stack
- Next.js 14 + TypeScript
- Tailwind CSS + Framer Motion
- Zustand + next-intl
- Axios + Socket.io

### Features
- ✅ Day/Night atmospheric modes
- ✅ Telegram-only authentication
- ✅ 4 languages (uz, uz-cyrl, ru, en)
- ✅ Real-time chat & notifications
- ✅ Premium animations
- ✅ Responsive design

### Project Structure
```
app/              # Pages
components/       # UI components
lib/              # Logic & utilities
messages/         # Translations
public/images/    # Assets
```

### Design Rules
- NO emojis (use Lucide icons)
- NO AI language (human moderation only)
- NO raw translation keys
- Premium Airbnb/Stripe quality
